## Contribution

The project follows MVVM architecture, and powered by

- Dagger 2
- ViewModel
- LiveData
- Data Binding
- Retrofit
- OkHttp (only for getting result image)

If you've basic knowledge in above stack, feel free to create PR to anything (features/bug fixes/anything)
